# Checkpoints

Download the pretrained model checkpoints from the Google Drive folder:

- [Download checkpoints from Google Drive](https://drive.google.com/drive/folders/1rQ4Z2zYd6ublf2TVP5Q8ElvX_MCxCtiF?usp=sharing)

After downloading, place the checkpoint files into this repository's `checkpoints/` directory so training or inference scripts can find them.
